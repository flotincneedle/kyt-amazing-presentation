# Cross-Validation Prompts

Before major milestones, we send our plans to Gemini and ChatGPT for independent review. This catches blind spots.

## How to use
1. Copy the prompt
2. Paste into Gemini (latest model) and ChatGPT (latest model)
3. Collect feedback
4. Bring back to Nexi for analysis and roadmap updates

## Prompts
- `01-architecture-review.md` — review of tech stack and architecture decisions (create after ARCHITECTURE.md is ready)
